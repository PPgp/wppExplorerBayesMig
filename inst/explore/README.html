Description

This is a modified version of the WPP Explorer that allows you to interactively explore probabilistic population projections with migration uncertainty produced using the method by Azose et al. (2016) contained in the R package wpp2015BayesMig. It is based on the shiny R package.

Data

Population and migration projections were produced using the Azose et al. (2016) methodology. All historical estimates as well as projections of fertility and mortality are from the World Population Prospects 2015, the United Nations, Department of Economic and Social Affairs, Population Division. See also our project website. Note that the projections provided here are not official UN projections.

Navigation

There is a time and indicator selection on the left, and various forms of visualization on the right. The time slider in the left pane defines the year or time period for which data are shown. Note that for vital rate indicators, such as fertility, mortality, migration and sex ratio at birth, the time shown on the slider is the end year of the time period of the data shown in the right pane.

The uncertainty selection is only used for indicators and years for which the corresponding lower and upper bounds are available, and for the appropriate visualization types. These are country plots that appear after clicking on the map, table in the “Sortable Data” tab, and “Trends” and “Pyramids”. For Population indicators, uncertainty is available for total population and for single age and sex groups. It is not available for total sex-specific population or if multiple age/sex groups are summed together.

The "Trends & Pyramids" tab can be used to compare indicators by countries. Selection of multiple countries is supported. The countries are sorted alphabetically. Various aggregated regions can be found at the end of the list. Thus countries can be compared to regional agregates in the same graph.

The ggplot-based graphs in the "Trends & Pyramids" sub-tabs are interactive. One can see the data values by mouse hovering or single clicks. One can zoom in by selecting a region of the graph. Zooming out is done by a double click.

Values in the "Trends" plots as well as in the table below each plot correspond to the indicator selected on the left. The values in the "Age Profiles" and "Pyramids" graphs correspond to the sex- and age-specific version of the selected indicator. For example, if "Total Population" is selected in the left panel, the "Age Profiles" and "Pyramids" graphs show population by sex and age, while the table below shows values of the total population.

The Probability Calculator can be used to compute probability that total population of the selected country will be larger or smaller than the given threshold. Results are shown for future 5-year periods until 2100. In the online version of the Explorer, the calculation is performed using bayesPop trajectories stored on the server. If run locally, a bayesPop simulation directory can be passed to the wpp.explore function as an argument.

The Rosling Chart is based on Hans Rosling's gapminder. After opening the tab, the x and y axes are preloaded with the first three indicators from the left panel. You can add indicators to the axes by selecting it on the left and clicking the "Add Indicator" button in the right panel. Age-specific indicators can also be selected and added to the chart. The chart is created via the R package googleVis which for the rendering requires an Internet connection and a Flash player. See the googleVis documentation for more details.
